# 啟動方式說明

## start.bat (單一窗口模式 - 推薦)
- **功能**：在一個窗口中顯示所有服務的輸出
- **優點**：方便監控所有服務的日誌，輸出清晰
- **注意**：**此窗口不能關閉**，關閉會停止所有服務
- **停止服務**：按 `Ctrl+C` 或關閉窗口

## start-background.bat (多窗口模式)
- **功能**：在分離的窗口中運行服務
- **優點**：可以關閉啟動窗口，服務繼續運行
- **注意**：會打開多個窗口（伺服器窗口、客戶端窗口）
- **停止服務**：需要關閉各個服務窗口，或使用 `stop.bat`

## 選擇建議
- **開發/測試時**：使用 `start.bat`（單一窗口，方便查看日誌）
- **日常使用時**：使用 `start-background.bat`（可以關閉啟動窗口）

## 端口說明
- 伺服器（後端 API）：http://localhost:3001
- 客戶端（前端）：http://localhost:3000
- 瀏覽器會自動開啟 http://localhost:3000
