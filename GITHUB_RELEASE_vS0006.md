## 🎯 主要更新

### ✨ 新功能

#### 1. 智能價格獲取策略
- 根據交易時間自動切換數據來源（盤中即時 / 盤後收盤價）
- 即時 API 失敗時自動降級到收盤價
- 支援手動選擇價格來源模式（自動/即時/收盤價）
- 庫存頁面顯示價格來源標籤和更新時間

#### 2. 使用指南頁面改進
- 新增讚賞碼 QR Code 圖片顯示（微信、富邦、全支付）
- 使用指南只在初次登入時顯示一次
- 圖片尺寸優化（原始尺寸的一半，更易掃描）

#### 3. 登入頁面改進
- 默認管理員帳號提示只顯示一次
- 可手動關閉提示
- 修改帳號/密碼後自動隱藏提示

#### 4. 後台管理改進
- 管理員設定中顯示默認帳號提示
- 更新帳號/密碼後自動標記提示為已看過

## ⚠️ 注意事項

1. **價格數據**：盤中時段使用即時價格（可能有延遲），盤後使用收盤價
2. **使用指南**：首次登入會自動跳轉，已看過後不會自動顯示
3. **默認管理員**：建議首次登入後立即修改帳號和密碼
